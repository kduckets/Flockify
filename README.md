#Flockify

##Prerequisites
* NodeJS ([https://nodejs.org/en/download/](https://nodejs.org/en/download/))
* nodemon
```
npm install -g nodemon
```

##Installation
1. Install node modules
```
npm install
```
2. Run

development (uses test database for local development)
```
npm run start-dev

```
--or--

production
```
nodemon server.js
```
